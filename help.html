What is this?

Todo App is a simple and beautiful web app for creating and managing todo lists.

How does it work?

It's built in HTML5 and Javascript, using local storage to save your todos in the browser instead of in a web server, which makes the app availible offline aswell.

Why doesn't it work?

Are you using Internet Explorer? No? Try pressing here. This should reset the Todo App, and make it functional again. However, all your current todos will be lost. Sorry.

Who created this Todo app?

Todo App is created by Filip Stefansson at Pixby Media. You can find him on Twitter.
